Class Manager

Singleton which manages state of all known paths. It provides two functionalities: 1) Executes Operations to either cache the resulting State in conjuction with its query or invalidate its dependent paths, and 2) Accepts subscriptions to cached State via queries and notifies relevant subscribers whenever cached State change.
